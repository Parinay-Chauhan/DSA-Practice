# DSA Practice

This repository contains my solutions to Data Structures and Algorithms (DSA) problems solved across multiple coding platforms. The solutions are organized topic-wise for easy navigation and revision.

## Platforms

- LeetCode
- Coding Ninjas
- GeeksforGeeks
- HackerRank

## Topics

- Arrays
- Strings
- Linked List
- Stack
- Queue
- Trees
- BTS
- Graph
- Binary Search
- Heap
- Recursion
- Dynamic Programming
- Bit Manipulation
- Greedy
- Backtracking

## Repository Structure

Each topic contains:
- Well-organized solution files
- Problem information
- Time and Space Complexity
- Approach used

## Note

The solutions are stored in the same format as they were submitted on the respective coding platforms.

Standard headers (`#include`) and the `main()` function are intentionally omitted because they are provided by the online judge.

This repository is intended for learning, revision, and interview preparation rather than as a collection of standalone executable programs.
